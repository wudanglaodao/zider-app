# Interactive Custom Cursor PRD

## Product Name

Interactive Custom Cursor

## Subtitle

```text
Interactive Cursor Experience for Wix
```

## Product Positioning

Interactive Custom Cursor is a Wix app that improves the interactive feel of a website through custom cursor behavior, hover motion, and click feedback.

It is not just a cursor replacement tool. The product goal is to make Wix sites feel more polished, branded, and interactive with minimal setup.

## Target Users

### Wix Website Designers

Needs:

- Improve perceived site quality
- Add common high-end interaction patterns quickly
- Avoid custom code

### Creative Studios and Agencies

Needs:

- Branded visual experience
- Portfolio and landing page interaction polish
- Reusable setup across client sites

### SaaS and AI Websites

Needs:

- More modern product feel
- Subtle motion feedback
- Better landing page interaction

### Photography and Portfolio Sites

Needs:

- Hover labels such as VIEW or OPEN
- Better gallery browsing feedback
- More immersive presentation

### Ecommerce Brand Sites

Needs:

- Product card hover feedback
- More premium shopping interaction
- Click and hover affordance

## V1 Product Goals

V1 should deliver:

- Full-site cursor system
- Link and button hover states
- Click pulse effect
- Dashboard configuration
- Simple presets
- One-time install and global effect

The user experience should be:

```text
Install app
↓
Open Cursor Settings
↓
Choose style and colors
↓
Apply to site
↓
Cursor works globally
```

## Cursor States

Interactive Custom Cursor should support these states conceptually:

```text
Default
Hover Link
Hover Button
Hover Image
Hover Product
Hover Video
Click
Loading
Disabled
```

MVP should implement only:

- Default
- Hover Link
- Hover Button
- Click
- Disabled

Other states can be reserved for later.

## Cursor Types

### Free / MVP

#### Dot

A small filled dot cursor.

Best for:

- Minimal websites
- SaaS pages
- Brand sites

#### Ring

A hollow circular cursor.

Best for:

- Creative sites
- Portfolio sites
- Visual landing pages

#### Dot + Ring

Two-layer cursor:

- Dot follows the pointer quickly
- Ring follows with smooth easing

Best for:

- Premium interaction websites
- AI and technology landing pages
- Design-focused homepages

### Reserved for Pro / Later

These are not MVP:

- Glass Cursor
- Image Cursor
- Text Cursor as a full cursor type
- Blend Cursor
- Magnetic Button
