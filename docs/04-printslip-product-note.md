# PrintSlip Product Note

## Product Name

PrintSlip

## Tagline

```text
Print custom invoices and packing slips for your store orders.
```

## Product Type

PrintSlip is a future Wix Stores operations app.

It is separate from Cursor Studio and should not be included in Cursor Studio MVP.

## Product Positioning

PrintSlip helps store owners customize and print order documents.

Main documents:

- Invoice
- Packing slip

Possible later documents:

- Return slip
- Shipping note
- Gift receipt

## Target Users

### Wix Stores Merchants

Needs:

- Print order documents quickly
- Add store branding
- Control visible order fields
- Make fulfillment workflow cleaner

### Small Ecommerce Teams

Needs:

- Batch print packing slips
- Avoid manually editing invoices
- Keep documents consistent

### Brand Stores

Needs:

- Branded invoice layout
- Branded packing slip layout
- More professional post-purchase experience

## Future MVP Scope

Potential MVP features:

- Invoice template
- Packing slip template
- Logo upload
- Store name and address
- Brand color
- Field visibility toggles
- Order selector
- Print preview
- Browser print support

## Out of Scope for Early PrintSlip

Do not include initially:

- Complex drag-and-drop editor
- Multi-language template marketplace
- Shipping label generation
- Accounting integrations
- Advanced tax logic
- Full PDF infrastructure unless required

## Technical Direction

PrintSlip likely needs:

- Wix Dashboard page
- Wix Stores order access
- Template rendering
- Print preview
- Optional PDF/export later

Unlike Cursor Studio, PrintSlip probably does not need:

- Front-site embedded script
- Cursor runtime
- Hover/click effects

## Relationship to Cursor Studio

Shared later:

- Zider brand
- Dashboard visual language
- Billing system
- App install registry
- Analytics

Separate:

- Runtime logic
- Product use case
- User intent
- Wix API surface
- App Market listing

printslip

## Planning Status

PrintSlip is reserved for later.
