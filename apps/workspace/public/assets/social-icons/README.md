# Social Icons

Common monochrome SVG icons for PrintOps templates.

Use public paths like:

```txt
/assets/social-icons/instagram.svg
/assets/social-icons/facebook.svg
/assets/social-icons/x.svg
```

Most icons are from Simple Icons and were downloaded as single-color `#111111` SVGs. LinkedIn is from Bootstrap Icons because the Simple Icons CDN did not provide a LinkedIn SVG at download time.
